```text

任务：新客户开发介绍第1/4封。
目标：第1封：说明身份和来意，确认是否找对负责人。
节奏：第 1 天发送：建立联系或恢复联系，先让客户理解这套邮件的业务价值，不急于成交

结构脚本：
第1封：给老客户公司新联系人发送第一封邮件，通过过往合作建立自然连接，唤醒客户记忆，并引导后续沟通或转介绍

【正文结构，建议 4 段】
开头：自然破冰，必须像人工邮件
• 直接称呼：{customer_name}
• 用我们曾经给客户提供过笔译服务做切入点
写“我们曾参与/支持过客户公司的相关项目”
• 语气要求：像熟悉客户业务的人写邮件，而不是介绍公司

第二段：选一个业务做切入
从以下业务中只选择一个方向展开：
•行业活动 / 展会支持
•双语资料 / 多语言内容
•会议口译 / 商务沟通支持
•对外宣传 / 国际展示内容
•视频 / 多媒体制作
•商务物料 / 礼品及本地化支持
要求：
先写一个具体业务场景（客户可能会遇到的情况）
用自然带出我们可以提供的支持方式
再用合同案例库中对应业务的案例，做我们优势的补充说明
重点表达：
1）我们经验丰富
2）协助客户减少多供应商沟通成本
3）帮助客户提升执行效率
4）使客户工作作更顺畅”
不要写成我们的提供服务的清单形式
不要写成我们提供翻译、设计、视频服务这样空洞的句子

结尾：
亲和的低压力收尾
再次感谢客户
用婉转客气的请气请客户帮忙转介绍
结尾不要出现：
1）提供免费试译
2）提供解决方案
3）预约会议

AI指令
必须使用 {history} 中的真实合作事实作为唯一信任基础
不允许虚构合作内容
不使用“最近”“回想起”“上次沟通”等表达
不把翻译/笔译当主卖点
主线必须围绕：{industry} 的国际化沟通或对外业务支持
每封邮件只聚焦一个 business_lines 切口
风格要求：像资深销售写给熟悉的客户公司新联系人，而不是营销模板

变量取值：
{customer_name} = 萧小姐
{company_name} = 孚乐率
{industry} = 工业设备/制造客户
```
