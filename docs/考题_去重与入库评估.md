# 考题去重与入库评估

## 结论

- `docs/考题知识库.csv` 整体价值高于邮件原始数据，适合整理为正式知识入库准备表。
- 但其中 `拷贝不走样` 类训练说明更像练习题，不适合直接作为正式检索知识，已剔除。
